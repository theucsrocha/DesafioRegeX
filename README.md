📋 Desafio: Analisador de Letras (MusicaService)

Projeto em Groovy para análise linguística da música Construção de Chico Buarque, focado em Regex de alta precisão e TDD.
🛠️ Funcionalidades

    Filtro de Caracteres: Lista palavras com ç, ã ou õ.

    Encontros Vocálicos: Identifica Ditongos, Tritongos e Hiatos.

    Frequência de Frases: Mapeia e conta frases repetidas.

    Métrica de 4 Palavras: Regex exclusivo para capturar exatamente 4 palavras no início de cada linha, tratando acentuação Unicode.

🚀 Tecnologias

    Linguagem: Groovy / Java.

    Testes: Spock Framework (TDD).
